This is a simple **Random Quote Machine** built using React and Tailwind CSS. The application displays a random motivational quote and allows users to share the quote directly on Twitter.

## Features
- Displays random motivational quotes.
- Responsive design with a clean UI.
- Allows users to tweet the current quote with one click.
- Dynamic color schemes for each quote.

---

## Technologies Used
- **React**: For building the user interface.
- **Tailwind CSS**: For styling and responsive design.
- **Font Awesome**: For the Twitter icon.
- **JavaScript (ES6)**: For application logic.
